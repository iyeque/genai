# Algo Agent Workspace

This directory contains the working files for the **Algo** sub-agent (ALGOL 26 language design and migration specialist).

## Structure

- `workspace/` - Algo's working area (scratchpad, drafts, logs)
- `BOOTSTRAP.md` - Initial instructions and mission
- `TASK.md` - Current task list (generated by Algo)
- `SPEC/` - Drafts of ALGOL 26 specification
- `TOOLCHAIN/` - Implementation of parser, interpreter, compiler
- `MIGRATION/` - Tools for ALGOL 68 → ALGOL 26 conversion
- `LOGS.md` - Daily progress log (to be created by Algo)

## Usage

Algo should:
1. Keep all work-in-progress in `workspace/`
2. Move finalized documents to appropriate repository locations:
   - Spec → `ALGOL_26_SPEC.md` at root or `docs/architecture/`
   - Tooling → `tools/algo/` or separate `compiler/` directory
   - Migration tools → `tools/migration/`
3. Update `LOGS.md` daily with progress and blockers
4. Report to parent (GenAI) via commit messages or `genai-memory/` updates
