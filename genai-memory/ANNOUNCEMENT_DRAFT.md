## Why ALGOL 26?

ALGOL 26 is the culmination of 50+ years of language design, from ALGOL 68 through modern Rust, Haskell, and probabilistic languages. It's:

- **Expressive** — algebraic types, pattern matching, dependent types
- **Safe** — ownership + optional GC, formal verification annotations
- **Concurrent** — built-in async/await, channels, distributed memory model
- **Probabilistic** — native `sample`, `factor`, `causal` blocks for reasoning under uncertainty
- **Hardware-aware** — `kernel` blocks for GPU/TPU execution
- **Self-modifying** — code as data, runtime code generation, meta-programming

In short: **the one language powerful enough to write AGI**.
